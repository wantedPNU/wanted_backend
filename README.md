## Setting
- pipenv 사용
- Python 3.10.9
- FastAPI
- MongoDB


## How to Start server

1. pipenv 설치 ($ pip install pipenv )
2. $ pipenv --python 3.10 (없을 시에 다운로드 받아라고 뜸)
3. $ pipenv install
4. $ sudo -u mongodb mongod --config /etc/mongod.conf 
5. $ pipenv shell
6. $ pipenv run start
